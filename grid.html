Bootstrap's grid system is built with flexbox and allows up to 12 columns across the page.
The Bootstrap grid system has five classes,These classes are just like the limits(Width), after these limits the div will take the full width.
Class "Row" ko "display:flex" lagta hai, so inside row all boxes will be inline. Row ko hum "COL's" mein bi divide kar sakte hain nahi to apni khud ki width bi de sakte hain.
NOTE: COL AUR WIDTH KABI BI SATH NAHI LAGENGE. "COL" bi ek tarah ki width hai jise hum "px" ya "%" mein nahi dete per div ko laga sakte hai without row. "Col" class will always be under the "row" class.

Ek hi row mein agar sirf 2 ya 3 col dikhane hain to row class ke sath .row-cols-2/3 use kare. Baki columns ultimately niche a jayenge equall width mein.
.col- (extra small devices - screen width less than 576px) = Mobiles
.col-sm- (small devices - screen width equal to or less than 576px) = Mobiles with bigger screen
.col-md- (medium devices - screen width equal to or less than 768px) = Tabs
.col-lg- (large devices - screen width equal to or less than 992px) = Small Laptops
.col-xl- (xlarge devices - screen width equal to or greater than 1200px) = Laptops
By default padding in the col classes are 15px from left and right.
To align the div's horizontally just use these classes in row class:-justify-content-between, justify-content-around,justify-content-end,justify-content-center

To skip some cols, use offset-3 , offset-md-3 classes for background
.bg-primary,.bg-secondary,.bg-success,.bg-warning,.bg-danger,.bg-info,.bg-dark,.bg-light.

Classes for text
.text-primary,.text-secondary,.text-success,.text-warning,.text-danger,.text-info,.text-white

Classes for the alignment of text
text-left,text-right,text-center,text-justify,text-uppercase,text-lowercase,text-captialize We can also use them with breaks

Classes for button
btn-primary,btn-secondary,btn-success,btn-warning,btn-danger should be used with "btn"
e.g. 'input type="submit" class="btn btn-primary btn-lg btn-sm" name="submit"'=>
